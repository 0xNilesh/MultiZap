Flow
1. Frontend - swap
2. Alice - maker token, taker amount - sign, approve -> relayer backend (api)
3. Order store in relayer, secret generate - starts dutch auction
4. Relayers - get orders (api)
5. Resolver calls api (bid auction) -> auction price fix, relayer success, 
6. auction -> order me winning Resolver set hoga
7. Resolver calls source chain escrow factory -> deploy a htlc escrow contract -> alice tokens 
8. Resolver calls dest chain escrow factory -> deploy a htlc escrow contract -> sends taker asset
9. Resolver calls relayer api -> secret reveal -> src, dest escrow -> verify -> return secret
10. Resolver call src, dest to unlock tokens
